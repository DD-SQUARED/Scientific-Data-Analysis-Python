The projects simulate real-world analytical workflows, demonstrating the ability to:

## Featured Project

### ⭐ Atmospheric Extinction Analysis (Recommended Project)
**Objective:**  
Analyse the relationship between instrumental magnitude and airmass to determine atmospheric extinction coefficients across multiple optical filters.

**Highlights:**
- Data cleaning and preprocessing of observational datasets
- Linear regression modelling for each filter (B, V, R, I)
- Extraction of extinction coefficients and zero points
- Strong correlation analysis with error estimation
- Clear visualisation of trends and model fit

**Why this project stands out:**
- Combines data analysis + statistical modelling
- Demonstrates real scientific problem-solving
- Closest to industry-level analytical work
